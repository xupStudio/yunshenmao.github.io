# 雲深貓園 · 認養申請表（Google 表單草稿）

這份草稿是給師父和網站維護者建立 Google 表單用的「題目清單」。建立步驟：

1. 到 https://docs.google.com/forms 開新表單
2. 表單標題：**雲深貓園 · 認養申請表**
3. 表單說明（複製貼上）：

> 感謝您願意給山上的毛孩一個家。
> 請花 5–8 分鐘填寫這份表單，您填寫的內容只會被師父與網站維護者閱讀，不會公開於網站或社群。
> 送出後，師父會在 3–5 天內透過您留下的聯絡方式回覆。
> 山上沒有網路訊號的時段回覆會比較慢，請見諒。

4. 設定 → 一般 → **不要**勾「收集電子郵件地址（驗證）」（避免要求填表人登入 Google）
5. 設定 → 回應 → 將回應傳送到 Google Sheet（建議命名「雲深貓園 · 認養申請回應」）
6. 設定 → 回應 → **開啟「每次有新回應就寄送電子郵件通知我」**
7. 依下面題目逐區建立。每一區用 **新增區段** 分開（右側工具列的 = 符號）
8. 完成後，傳送 → **<>（嵌入）**，把 src 網址貼到 [app/adopt/page.tsx:13](../app/adopt/page.tsx#L13) 的 `GOOGLE_FORM_EMBED_URL`；傳送 → 連結，貼到 `GOOGLE_FORM_OPEN_URL`
